# Lab 2 - Windows File Management and Navigation

## Objective
- Practice navigating the Windows operating system and managing files using File Explorer and system settings.

## Lab Environment
- Personally built computer
- File Explorer

## Tools Used
- File Explorer
- File Explorer Option

## Tasks Completed

### 1. Create and Organize Files and Folders
- Created a new folder on the Desktop named "AplusLab"
- Created subfolders for organization (Images,Documents, Reice's Org)
- Renamed files and folders

### 2. Show Hidden Files and Folders
- Enabled "Show hidden files, folders, and drivers" in File Explorer Options
- Verified hidden system files became visible

### 3. Display File Extensions
- Disabled "Hide extensions for known file types"
- Confirmed file extensions were visible

### 4. Copy, Move and Delete Files
- Copied files between folders
- Moved files using drag and drop
- Deleted files and restored them from Recycle Bin

### 5.  Search for Files
- Used the search bar in File Explorer to locate files by name and type

## Steps Performed
1. Opened File Explorer
2. Navigated to Desktop and created AplusLab folder
3. Accessed File Explorer options from the see more/view menu
4. Enabled hidden files and file extensions 
5. Practiced file operations like copy, move, delete
6. Used the search bar to find specific files

## Results
- Successfully navigated the Windows file system and performed essential file management tasks.

## Key takeaways
- File Explorer is essential for managing system files and troubleshooting
- Viewing file extensions helps identify file types and potential security risks
- Knowing how to organize files improves efficiency

- Windows Navigation
- File management
- System configuration
- Troubleshooting basic file visibility issues
