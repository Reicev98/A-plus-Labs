# Lab 03 - Command Line Basics (Windows Command Line Interface - CLI)

## Objective
- Learn how to navigate the file system and manage directories using the Window Command Prompt.

## Lab Environment
- Personally Built Computer - Windows 11
- Command prompt(cmd)

## Tools Used 
- Command Prompt
- Basic CLI commands

## Commands Practiced
- "cd" -  Change directory
- "dir" - List contents of a directory
- "mkdir" or "md" - Create a new directory
- "rmdir" or "rm" -  Remove a directory

## Steps Performed

### 1. Open Command Prompt
- Pressed "Windows + R"
- Typed "cmd" and pressed Enter

### 2. Navigate the File System
- Used "dir" to view files and folders
- Used "cd" to move between directories
  -Ex: "cd Desktop", "cd AplusLab"

### 3. Create a New Directory
- Created a new folder named "Lab3Test"
  - > mkdir Lab3test

### 4. Navigate into the New Directory
- Entered the folder "Lab3Test"
  - > cd Lab3Test

### 5. Create additonal Directories
- Created subfolders
  - > mkdir Folder 1
  - > mkdir Folder 2

### 6. Verify Firectory Structure
- used "dir"

### 7. Remove a Directory
- Navigated out of the folder
 - > cd ..
- Removed the directory
 - > rmdir Lab3Test

## Results
- Successfully created, navigated, and removed directories using Command Prompt.
